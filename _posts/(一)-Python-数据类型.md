Python有五个标准的数据类型：

1.Numbers（数字）
````
       int（有符号整型）如：100
       long（长整型）     如：-0x19323L
       float（浮点型）     如：15.20
       complex（复数）  如：45.j
````
2.String（字符串）     
```
      如:  'despairdog'
```
3.List（列表）      
```     
      如： ['I','golang',456]
```
4.Tuple（元组）     
```
       如： (789, 'golfing')
```
5.Dictionary（字典）    
```
   如：{'name': 'hong','age':17}
```
总结：
```
#好的，让我们一起简单总结下区别：
String（字符串）： '' 或者""
List（列表）: []
Tuple（元组）：（）
Dictionary（字典）：{}
```
